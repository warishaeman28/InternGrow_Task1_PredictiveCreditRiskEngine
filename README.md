# Predictive Credit Risk Engine

## 📌 Project Overview

This project was developed as part of the **Machine Learning Internship at InternGrow**.

The Predictive Credit Risk Engine uses machine learning classification algorithms to predict whether a loan application should be approved based on customer financial information.

---

## 🎯 Objectives

- Predict customer creditworthiness.
- Compare multiple machine learning classification models.
- Evaluate model performance using different metrics.
- Handle imbalanced datasets using SMOTE.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Imbalanced-learn (SMOTE)

---

## 🤖 Machine Learning Models

- Logistic Regression
- Decision Tree

---

## 📊 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
- Confusion Matrix

---

## 📂 Dataset

This project uses a **synthetic dataset** containing **1,000 customer records** created for educational and internship purposes.

### Features

- Age
- Annual Income
- Employment Years
- Credit Score
- Loan Amount
- Debt-to-Income Ratio
- Late Payments
- Savings

### Target

**LoanApproved**

- 1 = Approved
- 0 = Rejected
